# BizReach Playwright Test Automation

このリポジトリは、BizReachのウェブアプリケーションに対する自動テストをPlaywrightを使用して実装したものです。このプロジェクトでは、ユーザーがログインし、希望条件を設定するフローをテストしています。

## 特徴

- Playwrightを使用したE2Eテスト
- 自動ログインと希望条件の設定フロー
- CI/CDパイプラインに統合可能

## 技術スタック

- [Playwright](https://playwright.dev/)
- Node.js
- TypeScript
- Jenkins (CI/CD)
